# Format
## fixed.for

    1234567~72  
          +-+- body  
         +---- for line continuation symbol  
     +---+---- put any symbol to indicate for line continuation  
    +---+----- line numbers  
    +--------- C or * for comment  

## free.f90

    1~132  
    ; for new line (in the same line)  
    ! comment  
  
    conti& ! for line continuation, put & at last  
    nue
